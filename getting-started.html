Start your journey with CIN Framework by downloading the dedicated command-line tool for complete system management and control.

Download CIN CLI

Step 2: Install the Framework

Use CIN CLI tool to install the framework easily and quickly.

Framework Management Commands

Use these commands to efficiently manage CIN Framework:

Install Framework

To install CIN Framework, use the following command:

Terminal 
install framework [version]
Example: install framework 1.0.0

Update Framework

To update the framework to the latest version:

Terminal 
cin update framework

Delete Framework

To remove the framework from your system:

Terminal 
cin delete framework
