Complete Commands Guide

CIN CLI Commands

Complete guide to all CIN CLI commands for managing framework and libraries efficiently and easily.

Framework Management

Install Framework

cin install framework

Downloads and installs the latest version of CIN Framework (v2.0.0) with all core components.

Update Framework

cin update framework

Updates the framework to the latest available version while preserving current settings.

Remove Framework

cin delete framework

Completely removes the framework from the system with all its files and directories.

Library Management

Install Library

cin install library [library-name] [version]

Installs a specific library with a specified version. If no version is specified, the latest version will be installed.

Example: cin install library auth v1.2.0

Remove Library

cin delete library [library-name] [version]

Removes a specific library with a specified version from the system.

Example: cin delete library auth v1.2.0

List Installed Libraries

cin list libraries

Displays a list of all installed libraries with their versions.

Page Management

Create New Page

cin create page [page-name]

Creates a new PHP page with the basic CIN Framework template.

Delete Page

cin delete page [page-name]

Deletes a specific page from the project.

Utility Commands

Help

cin help

Displays a list of all available commands with a brief description of each.

Version Information

cin version

Displays the current CIN CLI version (v2.3.2) and system information.

Check for Updates

cin check-updates

Checks for available updates for CIN CLI and the framework.

Quick Start Guide

Basic Steps

Terminal
$ cin help

$ cin install framework

✓ Installing dependencies...
