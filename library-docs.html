CIN Library Complete Documentation

Complete guide to all components and utilities available in CIN Library with practical examples and installation instructions.

Installation & Setup

System Requirements

  • CIN CLI v2.3.2 or newer
  • CIN Framework v2.0.0 or newer
  • PHP 8.0 or newer

Library Installation

# Install CIN Library using CIN CLI
cin-cli library install

# Install specific library component
cin-cli library install component-name

# List available libraries
cin-cli library list

Available Components

UI Components

Card System

Customizable cards with diverse content support

<?php
// Using CIN Card Component
$card = new CIN\Library\UI\Card();
$card->setTitle('Card Title')
     ->setContent('Card content here')
     ->render();

Form System

Advanced forms with automatic validation and security

<?php
// Using CIN Form Component
$form = new CIN\Library\UI\Form();
$form->addField('email', 'email')
     ->addValidation('required|email')
     ->render();

Utility Components

Data Processor

Advanced data processing and formatting

<?php
// Using Data Processor
$processor = new CIN\Library\Utils\DataProcessor();
$result = $processor->process($data)
                   ->validate()
                   ->format();

Cache Manager

Advanced cache management with multi-level support

<?php
// Using Cache Manager
$cache = new CIN\Library\Utils\CacheManager();
$cache->set('key', $data, 3600)
      ->get('key')
      ->clear();

API Reference

Core Classes

Base class for all library components

<?php
abstract class Component {
    public function render(): string;
    public function setConfig(array $config): self;
    public function getConfig(): array;
}

Practical Examples

Complete Example

<?php
declare(strict_types=1);

use CIN\Library\UI\Card;
use CIN\Library\UI\Form;
use CIN\Library\Utils\DataProcessor;

// Create a complete page with library components
$processor = new DataProcessor();
$userData = $processor->process($_POST)->validate();

$form = new Form();
$form->addField('name', 'text')
     ->addField('email', 'email')
     ->addValidation('required|email');

$card = new Card();
$card->setTitle('User Registration')
     ->setContent($form->render())
     ->addClass('registration-card');

echo $card->render();
