Installation

System Requirements

  • PHP 8.0 or higher
  • Web Server (Apache/Nginx)
  • MySQL/PostgreSQL Database (Optional)
  • Composer (for external dependencies)

CIN Framework is designed to work without external dependencies, ensuring local self-sufficiency.

Installation using CIN CLI

# Download CIN CLI from GitHub Releases
# Place cin-cli executable in your project directory

# Install CIN Framework
cin install framework

# Verify installation
cin version

This is the recommended method for installing CIN Framework.

Architecture

MVC Pattern

CIN Framework follows the Model-View-Controller (MVC) pattern with modern enhancements for performance and security.

project/
├── cin/                    # CIN Framework Core
│   ├── Core/              # Core system files
│   ├── Config/            # Configuration files
│   ├── DOMinator/         # DOM manipulation engine
│   └── Security/          # Security components
├── app/                   # Application files
│   ├── Controllers/       # Application controllers
│   ├── Models/           # Data models
│   ├── Views/            # View templates
│   └── Config/           # App configuration
├── public/               # Public web files
│   ├── assets/           # CSS, JS, Images
│   └── index.php         # Entry point
└── storage/              # Storage and cache
    ├── cache/            # Cache files
    └── logs/             # Log files

DOMinator Engine

Advanced DOM Processing

DOMinator is a DOM processing engine specifically developed for CIN Framework, providing superior performance and complete control.

<?php
declare(strict_types=1);

use CIN\DOMinator\DOMProcessor;

// Create DOMinator instance
$dom = new DOMProcessor();

// Process HTML with advanced features
$html = $dom->process([
    'template' => 'user-profile.html',
    'data' => [
        'user' => $userData,
        'settings' => $userSettings
    ],
    'security' => [
        'csrf' => true,
        'sanitize' => true
    ]
]);

echo $html;

API Development

RESTful API

<?php
declare(strict_types=1);

/**
 * CIN Framework API Controller
 *
 * Example API controller for RESTful services
 *
 * @package CIN Framework
 * @author Ayoub Alarjani (MAWI MAN)
 * @license Proprietary - All rights reserved to CIN Framework
 */

class UserApiController extends CIN\Core\ApiController
{
    /**
     * Get user information
     */
    public function getUser(int $userId): array
    {
        $user = $this->userModel->findById($userId);
        
        return $this->jsonResponse([
            'status' => 'success',
            'data' => $user,
            'timestamp' => time()
        ]);
    }
    
    /**
     * Create new user
     */
    public function createUser(): array
    {
        $data = $this->getJsonInput();
        $user = $this->userModel->create($data);
        
        return $this->jsonResponse([
            'status' => 'created',
            'data' => $user,
            'id' => $user['id']
        ], 201);
    }
}
